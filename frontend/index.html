About Me

안녕하세요, 저는 최현성 입니다.

24년 초에 학교에서 앱 개발 프로젝트에 참여하면서 서비스를 개발하는 것에 큰 흥미를 느껴 웹개발 파트를 공부하기 시작했습니다.
제 목표는 프론트엔드를 중점으로 두고, 백엔드 및 AI 모델쪽도 개발이 가능한 풀스택 개발자가 되어 다양한 서비스를 개발하는 것입니다.

Features

What I Do

HTML

HTML로 다양한 레이아웃을 배치하고 관리할 수 있습니다. 특히 flex와 grid를 적극적으로 사용해 레이아웃을 구성하려고 하고 있습니다.

CSS

자주 사용되는 css 태그와 속성에 대해 이해하고 있습니다. 사용자 경험 향상을 위해 반응형 레이아웃에 관한 연습을 하고 있습니다.

JavaScript

기본적인 문법과 간단한 DOM 조작에 대해 숙지하고 있습니다. JS의 이론적인 내용에 약해 유튜브나 블로그 등에서 이론에 대한 아티클들을 자주 읽으려 노력합니다.

ReactJS

useEffect, useState의 작동 원리를 간단히 알고 있고, API로부터 데이터를 받아와 바인딩시킬 수 있습니다. 역시 이론적인 내용에 약하다고 생각해 다양한 아티클들을 자주 읽으려 노력합니다.

TailwindCSS

대표적인 Utility-First CSS로, css파일과 html 파일을 왔다갔다 할 필요 없이 인라인으로 빠르게 컴포넌트들을 완성할 수 있는 라이브러리입니다. 개인적으로 가장 선호하는 css라이브러리입니다.

Styled Components

React에서 사용 가능한 CSS-in-JS 라이브러리입니다. 컴포넌트별로 스타일을 지정해두고 재사용할 일이 많을 때 효율이 극대화됩니다.

Github

가장 많이 사용하는 원격 소스코드 저장소입니다. git 및 github를 사용할 때 필요한 대부분의 명령어를 숙지하고 있습니다. 협업 컨벤션에 관심이 많습니다.

FastAPI

CRUD API를 쉽고 빠르게 개발할 수 있고, API 문서화를 자동으로 수행해주는 라이브러리입니다. GET, POST 메소드에 해당하는 API를 만들어본 경험이 있고, 필요에 따라 추가적인 공부를 진행하고 있습니다.

Firebase

Google에서 개발한 백엔드 서비스입니다. Firestore와 Firebase Cloud Message(FCM)을 깊게 사용해본 경험이 있습니다.
